    Project Description

    More Than a Music Player.

    A feature-rich music player showcasing modern frontend development and interactive user experiences.

    Pulse Player is a modern music player built using HTML, CSS, and JavaScript. Designed with a premium glassmorphism interface, it delivers a smooth and interactive listening experience through features such as playlist management, favorites, real-time search, shuffle and repeat controls, keyboard shortcuts, theme switching, and responsive design.
